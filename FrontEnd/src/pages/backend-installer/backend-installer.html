Backend Installer

Install the backend

Use the next commands to download the server: First clone the repositorie: git clone https://github.com/algomezmu/TFG.git Get inside the TFG folder. cd TFG Install the Modules/dependencies for the server (If you don't have installed node & npm, go to the end of this page)
  • cd Backend
  • npm install
 Before initializing the server, go to the folder "config" to the file "config" and edit all the parameter. For instant, modify this two files with the SSL certification and Key
  • private_key.key
  • ssl_certificate.cer
Init the server npm start To make the server persist after closing the terminal, use the next command; nohup npm start > output.log &

Install Nodejs & npm

To install the needed programs for the server to execute, you should follow the next commands for Ubuntu/Debian distributions:
  • sudo apt update
  • sudo apt install nodejs
  • sudo apt install npm

Install the sysstat package

Sar command is a command to get the network bandwidth
  • sudo apt-get install sysstat
